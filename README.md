# style-dev-boilerplate

樣式開發樣板

## Requirement

node.js v.8.11.x 以上

## Getting Started

專案安裝步驟

```shell
# 安裝
$ npm install # Or yarn install

# 功能指令
# 將 scss 編譯為 css
$ npm run dev

# production模式
$ npm run build
$ npm start
```
